A program for generating and saving passwords for websites on your device.

Used technologies: tkinter, tkinter(messagebox), random, pyperclip.

Tkinter is used to create the UI of the application.
tkinter(messagebox) to display informational messages and when checking for empty fields.
The random module for generating a password when clicking on the Generate password button.
In the program file, you can add separate requirements for the number of certain characters in the PASSWORD GENERATOR
fields.
pyperclip is used so that the generated password is immediately added to the clipboard.
All your data is saved at data.txt
